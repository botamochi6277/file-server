# Nodejs file server


```zsh
% node file_server.js -h
file_server.js <コマンド>

コマンド:
  file_server.js path  target directory path

オプション:
      --version  バージョンを表示                                         [真偽]
  -p, --port     port number                           [必須] [デフォルト: 8002]
      --help     ヘルプを表示                                             [真偽]
```
